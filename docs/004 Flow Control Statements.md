# Flow Control Statements

- Block/clause
    
    ```python
    for i in range(10):
        print(i)
    
    x = 10
    if x > 5:
        print("x is greater than 5")
    else:
        print("x is less than or equal to 5")
    ```
    
- **Conditional Statements**
    - if statement
    - if…else statement
    - if…elif…statement
